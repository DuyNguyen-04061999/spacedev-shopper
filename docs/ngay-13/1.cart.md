# Cart

# API

1. Lấy thông tin cart:

    GET: http://course.spacedev.vn/cart/v2

    (Authorization Bearer)

2. Thêm sản phẩm vào giỏ hàng, cập nhật số lượng:

    PATCH: http://course.spacedev.vn/cart/:productId

    (Authorization Bearer)

    Body:
    ```json
    {
        quantity: 3
    }
    ```
